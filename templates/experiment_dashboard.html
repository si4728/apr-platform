Experiment Framework

사전 작성된 4단계 시나리오 실험 파일들을 백엔드 백그라운드 스레드에서 직접 작동시키고 지연 시간과 데이터 흐름 통계를 관측합니다.

Automated Experiment Automation Framework

자동화 시나리오 원격 트리거링 & 실시간 로깅 모니터
QoS Level Test
QoS 0, 1, 2 에 따른 전송 신뢰도 및 통계적 E2E 지연 속도를 비교 테스트합니다.
Payload Size Test
메시지 크기 변동(100B~10KB)에 따라 발생하는 병목 현상 및 지연시간을 측정합니다.
Queue Stress Test
수백 개의 동시성 메시지를 쏟아내어 대기열(Backlog) 적체 상황에서의 Drop 카운트를 테스트합니다.
Schema Variation Test
다양한 가상 중첩/오류 스키마들을 실시간으로 유입시켜 스키마 분류 거리를 시뮬레이션합니다.
APR Validation Test
실시간 부하 매개변수에 맞춰 머신러닝 예측 기반 APR 정책이 동적으로 작동하는지 유효성 검증을 실시합니다.
