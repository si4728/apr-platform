Publisher Payload Metadata

발행자(Publisher)가 전송하는 센서 데이터의 기본 메타데이터 구조 기준입니다. 이 스키마에 부합하지 않는 데이터는 Unknown으로 분류됩니다.

metadata
  • topic: MQTT 발행 토픽 (예: iot/sensor/temp_001)
  • sensor_id: 센서 고유 식별자
  • payload_type: 데이터 유형 (defined_sensor, flexible_json 등)
  • timestamp: 데이터 발생 시각 (네트워크 지연 측정용)
  • seq: 메시지 시퀀스 번호 (누락/순서 역전 확인용)
  • data: 실제 센서 측정값 (숫자 또는 JSON 객체)
